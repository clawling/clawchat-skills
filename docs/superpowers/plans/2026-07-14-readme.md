> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Tech Stack:** GitHub Markdown, embedded HTML alignment, Shields.io badges, Hermes skills CLI

- The language selector, title, tagline, badges, and section navigation are centered.
- The website URL is exactly `https://clawling.com`.

- Each skill has its own `hermes skills install` command immediately after its description.
- Skill claims remain supported by the checked-in `SKILL.md` files.
